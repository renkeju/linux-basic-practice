第一章 Linux 概述¶

1.1 Linux 发行版¶

理解操作系统¶

现在在我们的日常生活中，软件为我们提供了各种各样的功能。比如发送邮件、访问Web网站、编辑视频或音频、编辑文档文件和管理日程安排等等。

这些软件无一例外都需要使用硬件（构成计算机的组件），其本身结构就非常复杂。另外，计算机硬件分为很多种类，其功能和性能也不尽相同。

操作系统（以下简称“OS”）更有效的利用硬件的功能，使多个开发者更容易开发应用程序。提供基础功能软件，让许多用户能够轻松的使用计算机。

优秀的操作系统提高了开发人员的开发效率，使用户能够花费更少的精力学习使用所需的功能。

代表的OS有Linux、Microsoft Windows、macOS等。

OS的作用¶

OS的作用¶

OS由一组具有多个角色的程序组成。内核提供最基础的操作系统功能。管理所有硬件方面的信息，包括CPU利用率、内存管理、外部设备管理、机箱管理和硬件中断。程序员和用户可以利用内核提供的服务来利用硬件资源。

应用程序种类¶

用户程序是指针对特定的用途而创建的软件，具体取决您的用途。使用操作系统的功能进行操作。应用程序一般都是OS中包含的程序包组，它是由程序供应商打包在一起，通过委托开发或自行开发等制作和发布的。

Linux操作系统构成¶

Linux操作系统由Linux内核、库和用户控制平台组成。

Linux 内核¶

Linux系统的内核，它是为OS提供最基本功能的程序。CPU的管理、内存的管理、进程管理等等。OS的启动是从Linux内核被读取到引导系统开始的。

低功耗双模块¶

在系统启动后，根据需要从磁盘读取到内核地址空间的内核模块。与各种厂商不同的网络硬件相对应的驱动器（控制硬件的程序）等，作为低功耗双模块来提供。

库¶

库作为开发程序时所需函数的素材的程序。在 Linux 中，提供了GNU开发的库，以及X.Org开发的X库。

X Windows System¶

提供了由X.Org开发的X服务和X客户端构成的X Window System软件。在X Window System 上，桌面环境以统一的设计和操作作为基础，提供了菜单、图标和后退回台窗口等。

程序开发环境¶

作为解释程序，可以提供Bash、Python、Perl语言等，作为编译器，可以提供C、C++、Java开发环境等。

服务程序¶

在Linux上可以提供各种常驻服务。例如，有ssh交换通讯的程序、提供打印机服务的程序等。

命令、实用程序¶

在桌面环境中，有提供文字处理和表格计算等办公工具（LibreOffice），具有高级功能图形软件（GIMP）、Web浏览器、邮件工具、系统管理工具等。另外，还提供了一般用户命令和用户管理、网络管理、数据库管理的管理命令等。

什么是发行版¶

1991年，自李纳斯.托瓦兹在网上公开了第一版Linux内核以来，Linux内核就通过互联网吸引了许多开发人员参与并发展壮大。

除了Linux内核，还需要从源代码通过比较生成可执行文件，程序可使用的程序库，用户接口的Shell，需要许多软件才能将Linux作为完整的OS运行，包括从Shell启动的命令、工具，以及将它们安装到磁盘的安装程序。

集成这些软件，作为OS运行并发布（Distribute）的软件被称为**发行版**(Distribution或Distro)。

Linux操作系统有免费软件和开源软件组成。

自由软件¶

在GNU GPL（通用公共许可证）下分发的软件称为“自由软件”，该软件由GNU GPL（通用公共许可证）编写的。GPL的主要内容包括：

  • 二进制（执行程序。如果要发布编译和生成源代码），则该源代码也必须公开。

  • 可自由开发、变更、发布和使用

  • 以GPL发布的软件为基础进行开发和变更的软件必须基于GPL进行发布

GPL的这一独特规定保证了您可以重复分发以共享和发展软件。大多数配置Linux的主要软件是在GPL下分发的免费软件。

开源软件¶

源代码公开，重新分发自由的软件一般被称为“开源软件”。在开源倡议（Open Source Initiative：OSI）中，根据“开发源代码定义”（The Open Source Definition：OSD），规定了开放代码的主要条件如下：

  • 允许重新分发

  • 和编译的程序一起公开源代码

  • 改变后的软件，与改变前分配一样的许可证

这样，开源软件GPL规定变更了“GPL发布的软件为基础开发、软件一定必须在GPL的基础上发布”这样的项目。

如果您修改了在开源许可证中分发的软件，您也可以通过其他许可证分发。在某些情况下，对于想要隐藏源代码的公司来说，这是一个易于采用的许可。

X.Org开发的X Window System软件为“MIT许可证”，Mozilla组织开发的Web浏览器Firefox为“MPL（Mozilla Public License）”，构成Linux的一部分软件都已开放形式发布。

程序再造软件¶

程序再造软件是在不允许分发的情况下修改或复制的软件。发布者只提供信息，不公开源代码。一般情况下，得到程序再造软件是收费的。Microsoft Windows和在其上运行的收费软件是程序再造软件。

在一部分Linux发行版中，有时会包含程序再造软件。如果擅自将原程序软件复制散布，恐怕会涉及到著作权，所以需要注意。

这样，Linux的软件是通过免费软件和开源软件的许可证发布的，因此可以自由地进行重新发布和变更。因此，现在有RedHat Enterprise Linux(RHEL)、CentOS（RHEL下游发行版）、Ubuntu、Debian/GNU Linux等数百种发行版。

用户可以考虑自己的喜好和使用目的，从中选择合适的版本。这也是Linux最大的特征之一。
